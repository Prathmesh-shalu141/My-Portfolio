1. Understanding Versal ACAP and AI Engines

Versal ACAP (Adaptive Compute Acceleration Platform) is a heterogeneous platform by AMD Xilinx that integrates scalar processors, adaptable hardware (FPGA), and intelligent engines (AI Engines).

  • AI Engines (AIEs): SIMD/VLIW processors organized in a tiled array. They support high-throughput operations, ideal for matrix/vector computations.
  • Programmable Logic (PL): Provides fine-grained control and reconfigurability for custom hardware pipelines, memory buffering, and data routing.
  • AI Engines access data via shared memory or stream interfaces, and can be programmed using ADF graphs with kernels written in C++.
  • Versal ACAP targets applications in AI/ML, signal processing, 5G, automotive, and more.

2. Hands-on Work Using Vitis on Versal ACAP

As part of my research, I explored Vitis tutorials to gain practical experience on the VCK190 development board:

  • A to Z Bare-metal: Created simple complex arithmetic kernel and graph using 2023.2 Vitis flow.
  • LeNet AI Engine Design: Developed AIE-based image classification using memory-aware design. Validated on hardware.
  • GeMM Matrix Multiplication: Compared and designed in RTL/DSP/PS implementations for multiple matrix sizes up to 1024x1024.

I have also implemented some basic applications on AIEs using Vitis tool (AI Engine Kernel and Graph Programming )

  • Parallel Merge Sort: Used 11 AI Engine kernels to sort and merge 5000 numbers with shared buffer pipelining.
  • Custom Sum Computation: Computed sum of 5000 integers using 6 parallel AI Engines mapped across different tiles.
  • Matrix Multiplication Exploration: Studied shared memory, cascade connections, and kernel placement strategies.

All designs were tested using hardware emulation and verified on the actual VCK190 board via SD card deployment.

3. Exploration and Validation of Versal ACAP Frameworks

Analyzed multiple frameworks for AI acceleration on Versal ACAP including both academic research and open-source contributions:

  1. CHARM/CHARM 2.0: Composing Heterogeneous Accelerators for Deep Learning on Versal ACAP Architecture

    Accelerates matrix multiplications of diverse sizes. Applied in BERT, ViT, NCF, and MLP inference.
  2. High Performance, Low Power Matrix Multiply Design on ACAP: from Architecture, Design Challenges and DSE Perspectives
    Link to Paper
    AutoMM framework to generate efficient matrix multiplication designs on Versal ACAP.
  3. H-GCN: A Graph Convolutional Network Accelerator on Versal ACAP Architecture
    Link to Paper
    Acceleration of GNN inference using AI Engines.
  4. Exploiting On-chip Heterogeneity: of Versal Architecture for GNN Inference Acceleration
    Link to Paper
    GNN acceleration leveraging AI-PL-NoC synergy.
  5. Flexible Acceleration Framework for Dense/Sparse Matrix Multiplication: on Versal ACAP
    Link to Paper
    Python-based framework generating GEMM/SpMM source code across AI Engine, PL, host, and XRT.
  6. MaxEVA: Maximizing the Efficiency of Matrix Multiplication on Versal AI Engine
