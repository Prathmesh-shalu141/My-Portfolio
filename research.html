1. Understanding Versal ACAP and AI Engines

Versal ACAP (Adaptive Compute Acceleration Platform) is a heterogeneous platform by AMD Xilinx that integrates scalar processors, adaptable hardware (FPGA), and intelligent engines (AI Engines). It enables efficient acceleration of diverse workloads such as AI inference, DSP, and embedded compute.

  • AI Engines (AIEs) are SIMD VLIW processors organized in a tiled array. They support high-throughput operations, ideal for matrix/vector computations.
  • Programmable Logic (PL) provides fine-grained control and reconfigurability for custom hardware pipelines, memory buffering, and data routing.
  • AI Engines access data via shared memory or stream interfaces, and can be programmed using ADF Graphs with kernels written in C++.
  • Versal ACAP targets applications in AI/ML, signal processing, 5G, automotive, and more.

2. Hands-On Work Using Vitis on Versal ACAP

As part of my research, I explored Vitis tutorials to gain practical experience on the VCK190 development board:

  • A to Z Bare-metal Flow: Created simple complex arithmetic kernel and graph using 2023.2 Vitis flow.
  • LeNet AI Engine Design: Developed AI/PL-based image classification using memory-aware design. Validated on hardware.
  • GeMM Matrix Multiplication: Compared AI Engine vs RTL DSP58 implementations for multiple matrix sizes up to 1024x1024.

I have also implemented some basic applications on AIEs using Vitis tool

  • Parallel Merge Sort: Used 11 AI Engine kernels to sort and merge 5000 numbers with shared buffer pipelining.
  • Custom Sum Computation: Computed sum of 5000 integers using 6 parallel AI Engines mapped across different tiles.
  • Matrix Multiplication Exploration: Studied shared memory, cascade connections, and kernel placement strategies.

All designs were tested using hardware emulation and verified on the actual VCK190 board via SD card deployment.

Exploration and Validation of Versal ACAP Frameworks

Explored a range of frameworks designed for AI and matrix computation acceleration on Versal ACAP:

  • 1. CHARM/CHARM 2.0: Composing Heterogeneous Accelerators for Deep Learning on Versal ACAP Architecture
    Paper Link
    Accelerates matrix multiplication for both small and large sizes. Used in applications like BERT, ViT, NCF, and MLP inference.
  • 2. AutoMM: High Performance, Low Power Matrix Multiply Design on ACAP: from Architecture, Design Challenges and DSE Perspectives
    Paper Link
    Provides a framework for architecture-aware automated generation of matrix multiplication accelerators on Versal.
  • 3. H-GCN: A Graph Convolutional Network Accelerator on Versal ACAP Architecture
    Paper Link
    Accelerates GNN inference efficiently using the heterogeneity of Versal.
  • 4. GNN-Versal: Exploiting On-chip Heterogeneity of Versal Architecture for GNN Inference Acceleration
    Paper Link
    Designs heterogeneous GNN inference pipelines using AIE and PL elements.
  • 5. Dense/Sparse MM Framework: Flexible Acceleration Framework for Dense/Sparse Matrix Multiplication on Versal ACAP
    Paper Link
    Python-based source code generator targeting SpMM and GEMM acceleration using AIE and PL. Automates AIE/PL/host/XRT code.
  • 6. MaxEVA: Maximizing the Efficiency of Matrix Multiplication on Versal AI Engine
    Paper Link
    Efficient MM accelerator that improves tile utilization over CHARM.
  • 7. MaxEVA Adopted: Efficient Approaches for GEMM Acceleration on Leading AI-Optimized FPGAs
    Paper Link
    Builds on MaxEVA framework for improved GEMM throughput on Versal.
  • 8. WideSA: A High Array Utilization Mapping Scheme for Uniform Recurrences on ACAP
    Paper Link
    Maps uniform recurrences on AIE arrays to maximize utilization.
  • 9. AIM: Accelerating Arbitrary-precision Integer Multiplication on Heterogeneous Reconfigurable Computing Platform Versal ACAP
    Paper Link
    Supports very large integer multiplication (LIM, RSA, Mandelbrot) on AI Engine.
  • 10. PyAIE: Late Breaking Results: PyAIE: A Python-based Programming Framework for Versal ACAP Platforms
    Paper Link
    Provides a Python-based framework for AI Engine programming and rapid prototyping.
